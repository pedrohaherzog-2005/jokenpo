# Jokenpo

#### TADS e Engenharia de Software, 5 período, noturno

Essa é a entrega do produto 1, da matéria de Projeto de Aplicativo Mobile, que tem como objetivo criar uma aplicação no formato de um jogo em flutter chamado <strong>Jokenpo</strong> ou <strong>Pedra, Papel, Tesoura</strong>.

#### Membros/Contribuidores:

- Victor Caetano Da Conceição
- Pedro Henrique Alvarenga Herzog
- Debora Cupertino de Araújo Lacerda

#### Bônus:

E para agregar um pouco mais de valor ao projeto, vou criado um placar para contabilizar as vitórias e derrotas do jogador e da máquina, e também foi adicionado um modo de jogo chamado <strong>Melhor de Cinco</strong>.
